# Emotion_Detection
Streamlit application which detects the emotion from text using Go emotions dataset
Objective:
> user enters input text in text area
> Model predicts Emotion in text
> Display  original text and its confidence
> Top 3 Emotions
> visual representation
Technology:
> Machine learning models(Logistic regression,TF-IDF)
> Python
Tools:
> Pycharm for creating application
